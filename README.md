# Invisibility Cloak

A Harry Potter-inspired invisibility effect using **real-time image recognition and background substitution** with OpenCV.

## Tech Stack
Python · OpenCV · Image Processing

## Features
- Real-time color-based masking
- Background substitution for the "cloak" effect
